🎮 게임 개요
game6 하면 됩니다

목표:
플레이어 탱크를 조작하여 적들을 처치하고 점수를 올리며, 레벨을 올리고 보스를 물리치는 것.
적이 화면을 통과하면 게임 오버.

플레이 방식:

좌우 이동: ← →

점프: ↑

공격: SPACE

무기 업그레이드: Z (점수로 구매)

🛡 플레이어 (Player)

탱크 형태의 캐릭터

체력 (HP): 최대 100

무기 레벨: 레벨업 시 공격력이 증가하고 발사 딜레이가 감소

총알 발사: SPACE

업그레이드: 점수로 무기 강화 (Z)

피격 시 무적 시간 존재 → 깜빡이며 표시

체력 바 화면에 표시

👾 적 (Enemy)

일반 적 (Enemy)

화면 오른쪽에서 등장, 왼쪽으로 이동

체력과 이동 속도는 레벨에 따라 증가

일정 시간마다 플레이어 방향으로 적 총알 발사 가능

HP가 0이 되면 폭발 애니메이션 + 점수 획득

탱크형 적 (TankEnemy)

일반 적보다 체력 많고 느림, 점수 높음

발사 딜레이가 더 길고, 총알은 플레이어로 향함

HP가 0이 되면 폭발 + 점수 획득

보스 (Boss)

특정 레벨마다 등장 (BOSS_LEVEL_INTERVAL = 3)

이동 속도 느림, 공격력 강함, 체력 많음

플레이어에게 큰 피해 주는 총알 발사

체력 바 크게 표시, 처치 시 다음 스테이지 진행

💣 총알

플레이어 총알 (Bullet)

발사 시 x좌표 이동, 화면 끝까지 도달하면 삭제

무기 레벨에 따라 공격력 증가

적 총알 (EnemyBullet)

적이 플레이어 방향으로 발사

플레이어와 충돌 시 HP 감소

💥 폭발 (Explosion)

적 처치 시 폭발 애니메이션 표시

크기가 점점 커지며 사라짐

🏆 점수 & 레벨

적 처치 시 점수 획득

일반 적: 10점

탱크 적: 30점

보스: 100점

점수 기준으로 레벨업 → 적 난이도 증가

무기 업그레이드 비용도 레벨과 함께 증가

⚠ 게임 오버 조건

플레이어 체력 0

일반 적 또는 탱크 적이 화면 왼쪽 끝까지 통과

게임 오버 시 화면 중앙에 “GAME OVER” 표시

아무 키 입력 시 게임 종료

🖌 UI

화면 좌상단: 점수, 현재 스테이지, 다음 레벨까지 점수

화면 우상단: 무기 업그레이드 정보 (Z 키)

적과 플레이어 체력 바 표시

🔑 특징 정리

다양한 적 종류: 일반 적 → 탱크 적 → 보스

체력 기반 적: 총알을 맞출 때마다 HP 감소

적 공격 기능: 적 총알, 플레이어 피해

무기 업그레이드: 공격력 강화 & 발사 속도 증가

스테이지별 난이도 증가: 레벨업마다 적 HP/속도 증가

보스전: 일정 레벨마다 강력한 보스 등장
